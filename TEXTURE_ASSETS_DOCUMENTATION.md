# Документація текстурних пакетів проекту AzZiNnia

## Огляд текстурних ресурсів

Проект містить 6 основних текстурних пакетів, кожен з яких має своє призначення та характеристики. Ця документація детально описує кожен пакет, його вміст та можливості використання в грі.

## 1. 8K Skybox Pack Free

### Опис
Високоякісні skybox текстури для створення реалістичного неба. Пакет містить 12 різних варіантів неба в надвисокій роздільній здатності.

### Вміст
- **12 skybox матеріалів** (sky-1 до sky-12)
- Кожен skybox має текстури в форматі cubemap
- Приклади сцен для тестування

### Технічні обмеження
- Android: максимальний розмір cubemap - 2048
- iOS: максимальний розмір cubemap - 4096

### Використання в проекті
Ці skybox'и ідеально підходять для:
- Денних сцен з різними погодними умовами
- Вечірніх та ранкових пейзажів
- Створення атмосферних сцен української природи

## 2. Cartoon_Texture_Pack

### Опис
Стилізовані текстури в мультяшному стилі, які надають грі приємний візуальний вигляд. Це один з основних текстурних пакетів для нашого проекту.

### Структура
- **WOOD** - текстури дерева
- **WALLS** - текстури стін
- **SAND** - текстури піску
- **ROOF** - текстури дахів
- **ROCKS** - текстури каміння
- **GRASS** - текстури трави
- **DIRT** - текстури землі/бруду
- **_MaterialLibrary** - бібліотека готових матеріалів

### Переваги
- Візуально приємний стиль
- Добре поєднується з воксельною графікою
- Підходить для створення стилізованих українських сіл та природи

## 3. Cursors

### Опис
Набір RPG курсорів розміром 256x256 пікселів для різних ігрових ситуацій.

### Доступні курсори
- **Arrow** - стандартна стрілка
- **G_Cursor_Move1** - курсор переміщення (рекомендований як основний)
- **Cursor_Attack** / **G_Cursor_Attack** - курсори атаки
- **Cursor_Hand** / **G_Cursor_Hand** - курсори взаємодії
- **Cursor_Flask** - курсор для зілля/предметів
- **Cursor_loot** - курсор для збору предметів
- **Cursor_Production** - курсор для виробництва
- **Cursor_Settings** - курсор налаштувань
- **Cursor_ship** - курсор для кораблів
- **Cursor_shoot** - курсор для стрільби
- **Cursor_Deff** - курсор захисту

### Примітка
Курсори з префіксом "G_" мають золотистий відтінок, що може краще пасувати до української тематики.

## 4. Shaded Spectrum

### Опис
Текстури для створення містичної атмосфери, ідеально підходять для древніх цивілізацій та втрачених місць.

### Використання в проекті
- Скіфські кургани
- Древні святилища
- Таємничі підземелля
- Магічні артефакти

## 5. GreenMossGames - Ultimate Material Library

### Опис
Другий основний текстурний пакет з величезною кількістю реалістичних матеріалів. Містить професійні PBR текстури для різних поверхонь.

### Категорії матеріалів
- **wood** - різні види дерева
- **rock** - каміння та скелі
- **misc** - різноманітні матеріали
- **grass, sand and mud** - трава, пісок та грязь
- **concrete** - бетон
- **brick and stone** - цегла та камінь
- **asphalt** - асфальт

### Технічні особливості
- Підтримка Standard Render Pipeline (SRP)
- Можливість конвертації в URP/HDRP через включені пакети
- PBR текстури з normal maps та іншими картами

### Рекомендації
Рекомендується створювати копії матеріалів для налаштування:
- Tiling (масштабування текстур)
- Колір
- Інтенсивність normal map
- Інші параметри залежно від розміру об'єктів

## 6. YughuesFreeSandMaterials

### Опис
Спеціалізований пакет з різними видами піску для створення різноманітних регіонів України.

### Вміст
- 10+ різних видів піску (Sand01 - Sand10+)
- Кожен матеріал має normal maps
- URP пакет включено

### Використання для регіонів України
- **Південні регіони** - світлий пісок для степів
- **Прибережні зони** - морський пісок
- **Річкові береги** - річковий пісок
- **Пустельні зони** - для посушливих територій

## Рекомендації щодо використання

### Основні текстурні пакети
1. **Cartoon_Texture_Pack** - для основних об'єктів та будівель
2. **GreenMossGames** - для деталізованих природних об'єктів

### Додаткові пакети
- **8K Skybox Pack** - для атмосфери
- **Shaded Spectrum** - для містичних локацій
- **YughuesFreeSandMaterials** - для різноманітності ландшафтів
- **Cursors** - для покращення UX

### Поєднання стилів
Хоча Cartoon_Texture_Pack має стилізований вигляд, а GreenMossGames більш реалістичний, їх можна успішно поєднувати:
- Використовувати cartoon текстури для далеких об'єктів
- Реалістичні текстури для близьких деталей
- Змішувати стилі для створення унікальної естетики

## Технічні поради

1. **Оптимізація**: Використовуйте текстури меншої роздільної здатності для мобільних платформ
2. **LOD**: Застосовуйте різні текстури для різних рівнів деталізації
3. **Батчінг**: Групуйте об'єкти з однаковими матеріалами для покращення продуктивності
4. **Texture Atlasing**: Розгляньте можливість створення атласів для часто використовуваних текстур
